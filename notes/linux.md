**Navigation**
***
**1. Know the Current Path**
* pwd

**2. Know the Files and Directory**
* ls

**3. Change the Directory**
* cd

**4. Create a new Directory**
* mkdir

**5. Remove the Directory**
* rm

**6. Copy the files from one Directory to Another**
* cp

**7. Move the files from one Directory to Another**
* mv

***

**File Operations**
***
**1. Create a file**
* touch

**2.**
* View a file
    * cat
* View Multiple file
    * cat file_name file_name
* Create New Files
    * cat > file_name
* Append to Files
    * cat file_name >> file_name
* Show Line Numbers
    * cat -n file_name
* Display Non-Printing Characters
    * cat -e file_name

**3. Default Text Editor**
* nano 
***

**Search**
***
**Find the files in Current Directory**
* find
***

**Package Management**
***
**1. Update the Package Manager**
* sudo apt update

**2. Install the New Packages**
* sudo apt install

**Permissions**
***
**1. It's give permision in MODE**
* chmod

* sudo

**Process Commands**
***
**1. View Information about Currently Running Processes**
* ps

**2. Built-in Monitoring Tool**
* top

**3. Sending a Signal to that Process to Change its Behavior/ Killing the Process**
* kill

**Network Basics**
***
**1. Check if a Host is Reachable**
* ping (Packet Inter-Network Groper)

**2. Transfer Data to or from a Server**
* curl (Client URL)

**3. Download Files from the Web**
* wget (World Wide Web Get)
